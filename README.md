# Techno Brain | Front-End Developer Portfolio

![Project Screenshot](screenshots/desktop-view.png)

## Description
This is a professional and responsive front-end portfolio project for showcasing web development skills. Built using **HTML, CSS, and JavaScript**, it includes:

- Responsive navbar with mobile toggle
- Hero section with typing animation
- About section
- Services section with CSS Grid layout
- Contact form with validation
- Smooth scroll behavior

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6)
- Responsive design for mobile, tablet, and desktop

## Features
- Fixed header with smooth scroll
- Interactive and animated sections
- Mobile-friendly navigation menu
- Form validation with alert message
- Modern and clean UI design
